Shooting for the Stars

Overview

Shooting for the stars is a target shooting game inspired by retro arcade machines! This project was created for the Columbia ENGIE 1102 Art of Engineering class Game Showcase. This semester-long project included idea generation, prototyping, manufactuirng, and electronic integration.
